🔥 Pripojit Firebase
Firebase zajisti ze vsichni (Tatka, David, Pepa) vidi stejna data v realnem case. Nastaveni zabere 5 minut a dela se jen jednou.
1 Vytvor projekt
Jdi na console.firebase.google.com, klikni Pridat projekt, nazvi ho stanek-kalendar, vypni Analytics.
2 Zapni Realtime Database
Levy panel: Build → Realtime Database → Create database → europe-west1 → Start in test mode
3 Zkopiruj udaje
Project Settings (⚙) → General → Add app → Web → zkopiruj hodnoty do poli nize:
